생성형 AI 기반 회의록 요약 및 달력 인젝션 흐름도

대화 내용 송신
프롬프트 전송
요약 결과 반환
요약 데이터 전달
STEP 1. 사용자 화면

대화 기록 전송

사용자가 오픈채팅방에서 복사한 대화 텍스트 원문을 백엔드 서버에 전달하여 요약을 요청함

STEP 2. 백엔드 서버

기준 일자 계산 및 달력 구성

한국 표준시 기준으로 오늘의 정확한 날짜를 구하고, 환각 현상 방지용 요일 달력을 동적으로 조립하여 프롬프트에 주입함

STEP 3. GENERATIVE AI

안건 요약 및 시간 추론

주입된 기준 달력을 참고해 "다음주", "내일" 등의 상대 시점을 실제 날짜로 추론하고, 안건별 맥락을 분리하여 응답을 구성함

STEP 4. 백엔드 서버

데이터 파싱 및 정합성 검증

수신된 JSON 응답 데이터를 파싱하고, 미지정된 일정이나 정보 누락을 방지하기 위한 예외 가공 처리를 적용함

STEP 5. 사용자 화면

분석 일정 카드 가시화

백엔드에서 분석을 마친 일정, 회의 주제, 역할 분담 요약 데이터를 캘린더와 구조화된 카드 형태로 최종 출력함
